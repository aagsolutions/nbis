class TextField(text: String) : Field<String>

Represents a text-based field capable of storing and managing string data.

This class implements the Field interface to encapsulate text data, providing mechanisms to retrieve the data, its length, and create a deep copy of the field. It also implements basic equality and hash code functionality based on the text content.
