enum FingerprintFrictionRidgePosition : StandardReference, Enum<FingerprintFrictionRidgePosition>

NIST reference enum for friction ridge position codes.

Defines all the standard friction ridge positions used in NIST format for fingers, palms, and plantar surfaces according to various NIST standards.
