ImpressionType

enum ImpressionType : StandardReference, Enum<ImpressionType>

Represents the different types of impressions for biometric data as defined by various standards. Each type includes an identifying code, descriptive name, the standard in which it was introduced, and optionally the standard in which it was deprecated.
