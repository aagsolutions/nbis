CompressionAlgorithm

enum CompressionAlgorithm : StandardReference, Enum<CompressionAlgorithm>

Represents an enumeration of various compression algorithms that are compliant with ANSI/NIST standards.

Each compression algorithm is associated with a unique code, binary code, and description. The enumeration also tracks the standard from which each algorithm was introduced and, if applicable, the standard in which it became deprecated.
