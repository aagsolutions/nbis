Color

enum Color : StandardReference, Enum<Color>

Represents a set of predefined colors, each associated with a standard code, a description, and compatibility information regarding their creation and deprecation across specific NIST standards.

This enum implements the StandardReference interface, enabling compatibility validation for various NIST standards.
