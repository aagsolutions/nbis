SubjectFacialDescription

enum SubjectFacialDescription : StandardReference, Enum<SubjectFacialDescription>

Represents various facial descriptions for a subject, conforming to specific ANSI/NIST-ITL standards. Each description provides details about the subject's facial expression, pose, or accessories that may affect their appearance during biometric data collection.
