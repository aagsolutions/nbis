LatentImageRecordBuilder

class LatentImageRecordBuilder : NistRecordBuilder<LatentImageRecord, LatentImageRecordBuilder>

Builder class for creating and modifying Type-13 Latent Image records in NIST files.

This builder provides methods to set various fields specific to latent fingerprint images, including image capture information, processing parameters, and quality metrics. It supports the creation of records conforming to the ANSI/NIST-ITL 1-2011 standard.

The Type-13 record is used for storing and exchanging variable-resolution latent friction ridge image data, such as fingerprints, palm prints, and plantar prints.
