NistRecordBuilder

abstract class NistRecordBuilder<T : BaseRecord, B : NistRecordBuilder<T, B>>(val id: Int, val label: String, val calculator: LogicalRecordLengthCalculator)

Abstract builder class for creating and modifying NIST records.

This builder provides a base implementation for constructing different types of NIST records, managing their fields and ensuring proper record construction.
