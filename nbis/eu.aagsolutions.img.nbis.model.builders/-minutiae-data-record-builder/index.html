MinutiaeDataRecordBuilder

class MinutiaeDataRecordBuilder : NistRecordBuilder<MinutiaeDataRecord, MinutiaeDataRecordBuilder>

Builder class for creating and modifying Type-9 Minutiae Data records in NIST transactions.

This builder exposes fluent helpers for setting the fields defined in MinutiaeDataFields (ANSI/NIST-ITL 1-2011 and later). It automatically computes and injects the required LEN field during build().

Type-9 records contain minutiae, ridge counts and associated feature metadata for friction ridge impressions (finger, palm, plantar), including legacy and EFS elements.
