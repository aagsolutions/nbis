NistFile

class NistFile(recordsByType: Map<RecordType, List<BaseRecord>>)

Nist file container.

Holds an immutable map of all records grouped by RecordType and provides helpers to retrieve records by type and IDC, the transaction information record, and convenience getters for common record-type lists.
