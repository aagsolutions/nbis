enum PalmPrintImageFields : FieldType, Enum<PalmPrintImageFields>

Enum class representing various types of palm print image metadata and attributes.

The PalmPrintImage enum is specifically tailored to handle the unique information related to palm print images within a NIST record. Each entry in the enum corresponds to a distinct field, defining its attributes such as unique identifier, standardized code, description, and associated data type. This enum implements the FieldType interface, ensuring alignment with the expected behavior and metadata structure for field types.

Constructors:

  • One constructor allows the initialization of a PalmPrintImage instance based on another FieldType instance, inheriting its attributes.

  • Another constructor provides explicit declarations for field attributes such as id, code, description, and typeClass.
