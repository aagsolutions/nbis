StandardNistValidatorError

enum StandardNistValidatorError : Enum<StandardNistValidatorError>

Represents a standard NIST validation error.

This class encapsulates details of an error encountered during the validation of a NIST record, including a descriptive message, the type of the field where the error occurred, and an optional error code.
